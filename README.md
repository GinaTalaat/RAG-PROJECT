# RAG CV Matching System

## 📌 Project Idea

This project builds a Retrieval-Augmented Generation (RAG) system to match CVs with job descriptions.

## ⚙️ Features

* Upload multiple CVs (PDF)
* Chunking and embedding
* Vector search using FAISS
* Answer generation using LLM

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Upload CV PDFs
3. Run all cells

## 🧠 Technologies Used

* Python
* LangChain
* FAISS
* Sentence Transformers
